실험적 기능
기능 설명:
맥북에서 사용 중인 고품질 음성 설정을 추출하여 안드로이드에서도 비슷한 품질로 재생할 수 있도록 설정을 생성합니다.
맥북 음성 설정 추출하기
추출된 음성 설정:
{
  "preferredVoices": {
    "ko": [],
    "en": [
      {
        "name": "Karen",
        "lang": "en-AU",
        "voiceURI": "Karen",
        "fallback": "Google US English"
      },
      {
        "name": "Samantha",
        "lang": "en-US",
        "voiceURI": "Samantha",
        "fallback": "Google US English"
      }
    ],
    "zh": [
      {
        "name": "Sinji",
        "lang": "zh-HK",
        "voiceURI": "Sinji",
        "fallback": "Google 普通话（中国大陆）"
      },
      {
        "name": "Tingting",
        "lang": "zh-CN",
        "voiceURI": "Tingting",
        "fallback": "Google 普通话（中国大陆）"
      }
    ],
    "ja": [
      {
        "name": "Kyoko",
        "lang": "ja-JP",
        "voiceURI": "Kyoko",
        "fallback": "Google 日本語"
      }
    ]
  },
  "voiceSettings": {
    "rate": 0.9,
    "volume": 0.8,
    "pitch": 1
  },
  "fallbackStrategy": [
    "Google TTS (권장)",
    "Samsung TTS",
    "시스템 기본 TTS"
  ]
}
복사
다운로드
사용 방법:
1. 이 설정을 안드로이드 개발자에게 전달
2. Google TTS 앱을 설치하고 해당 언어팩 다운로드
3. 앱에서 이 설정을 기반으로 음성 우선순위 적용
